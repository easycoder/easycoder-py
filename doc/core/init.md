## Syntax:
`init {stack}`
## Examples:
``stack ValueStack`  
`init ValueStack`
## Description:
`init` initialises a stack - an array object, so that values can be [push](push.md)ed and [pop](pop.md)ped to and from it. Stacks are a convenient place to store values during recursive operations.
